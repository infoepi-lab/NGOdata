# Download IRS year index CSV (cached under `paths$index_dir`)

Download IRS year index CSV (cached under `paths$index_dir`)

## Usage

``` r
irs_download_index(year, paths = irs_project_paths())
```

## Arguments

- year:

  Tax index year

- paths:

  From
  [`irs_project_paths()`](https://infoepi-lab.github.io/irs-data/reference/irs_project_paths.md)
