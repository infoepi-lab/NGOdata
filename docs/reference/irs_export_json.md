# Export data frame to JSON file

Exports a data frame to a pretty-formatted JSON file in the specified
directory.

## Usage

``` r
irs_export_json(df, filename, output_dir)
```

## Arguments

- df:

  Data frame to export

- filename:

  Name of the output JSON file

- output_dir:

  Directory path for output file

## Value

Invisibly returns the full file path
